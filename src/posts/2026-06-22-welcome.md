working — markdown went in, a styled page came out.

## Writing a new entry

From the project folder, run:

```bash
npm run new -- "A title for your entry"
```

That creates a dated markdown file in `src/posts/` with the frontmatter already
filled in. Open it, write below the `---`, and save.

## Live preview while you write

```bash
npm start
```

This serves the site locally and reloads the page every time you save. Leave it
running in one terminal and write in your editor in another.

## Formatting

You have the full range of markdown:

- **bold**, *italic*, and `inline code`
- lists, like this one
- [links](https://www.11ty.dev/)

> Blockquotes are good for the lines you want to remember.

When you're happy, `npm run build` writes the finished site to `_site/`.
